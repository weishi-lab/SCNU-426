# SCNU-426
  该泡泡堂小游戏复现了经典4399小游戏泡泡堂，虽然只实现了部分功能，但由于时间关系，也没有继续实现完整的泡泡堂小游戏，这个项目只是我们大二的一个初级软件实训项目，不足之处敬请原谅。

控制方式
玩家一：向上移动 W
       向下移动 S
       向左移动 A
       向右移动 D
       空格键放置炸弹
玩家二：向上移动 ↑
       向下移动 ↓
       向左移动 ←
       向右移动 →
       右边数字键盘 0 放置炸弹
